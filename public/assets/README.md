# Public Assets

Place static image assets here. Files in this folder are served from the site root:

- `public/assets/icons/logo.png` -> `/assets/icons/logo.png`
- `public/assets/images/banner.gif` -> `/assets/images/banner.gif`

Suggested folders:

- `icons/` for favicon, app icons, logos, and small brand marks.
- `images/` for PNG/JPG/WebP/GIF content images.

For the website icon update, put the source icon files in `icons/`.
